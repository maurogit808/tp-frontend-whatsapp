# Aplicación de Mensajería – TP Final Frontend

Este proyecto es mi entrega del Trabajo Práctico Final de Frontend.  
Consiste en una aplicación web inspirada en WhatsApp, desarrollada con React, totalmente responsiva y desplegada online.

---

## Tecnologías utilizadas

- React
- React Router DOM
- CSS puro
- Vite (si corresponde)
- Context API (si se utiliza para tema o estados globales)

---

## Funcionalidades principales

- Lista de chats con último mensaje y hora.
- Vista de conversación con header del contacto.
- Buscador de chats funcional.
- Tema claro/oscuro.
- Navegación entre pantallas con React Router.
- Parámetros de búsqueda en la URL.
- Diseño responsivo (320px → 2000px).
- Accesibilidad: buen contraste y tipografías legibles.

---

## Rutas de la aplicación

- `/` → Página principal con lista de chats.
- `/chat/:id` → Chat seleccionado.
- `/search?query=` → Página de búsqueda (si aplica).

---

## Estados utilizados

- Chat seleccionado
- Búsqueda
- Lista de mensajes
- Tema claro/oscuro (si aplica)

---

## Formulario incluido

- Input de búsqueda de chats
- (Opcional) Formulario de envío de mensajes

---

## Principios aplicados

- DRY: componentes reutilizables
- KISS: estructura simple
- YAGNI: solo lo necesario
